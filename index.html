Конспекты занятий по HTML / CSS

1 занятие

Понятие - определение которое раскрывает смысл
Теги - бывают парные и не парные
Основные теги :
p - парный тег абзаца
h1 - парный тег заголовка
div - парный тег универсального контейнера
img - непарный тег добавления изображения

Для корректной работы тег img должены содержаться атрибуты :
src - путь к изображению
alt - альтернативный текст для изображения
lorem -  текст (к примеру), если добавить любую цифру рядом то текст будет на столько символов
index.html - правая кнопка мыши и 3 вкладка что бы посмотреть сайт
p*цифра - откроет столько р сколько надо  
hr - не парный тэг, горизонтальная полоса
sub - подстрочный
sup - надстрочный
br - прерывание
ColorSheme.ru - смотреть цвета, совместимость, писать через #
CSS - подключается в head : style поключается через link после head
text - alight - выравнивание текста
font - family - шрифты
Что то менять у всего текста - *{}
pre - парный тег как р,отбражет текст так же как в коде
span - парный тег, обертка для небольшого куска текста - выделение
font - weight - толщена текста
Web Gradients - рабочие градиенты сайт
back-ground color - цвет фона, если написать в р то фон текста изменится (CSS)
Коментарий в CSS - /*...*/
Отребуты - пишутся в открытом теге
class - название (* p class = "имя"/p )
В CSS - .имяclass{}
border - граница (рамка)
p { border : толщина 5 хр* тип: solid цвет }
del - парный тег, перечеркивание логического характера
s - парный простое перечеркивание
can i use.com и can i include.com - сайты поддержки тегов
b - делает жирным текст на сайте но не в поисковике
strong - делает и в тексте и в поиске
i - письменно на сайте, на сайте и в поиске - em
code - когда хотим написать кусок кода на С++ или Питоне
